# Healthcare Patient Conversion & Provider Performance Intelligence

## Project Overview

This project analyzes the complete patient journey to identify factors influencing treatment conversion, provider performance, communication effectiveness, and business revenue. Using data analytics, SQL, machine learning, and patient segmentation, the project provides actionable insights that support data-driven decision-making in healthcare operations.

---

## Problem Statement

Healthcare organizations receive numerous patient inquiries, but many patients do not complete treatment due to communication delays, provider performance differences, operational inefficiencies, and patient disengagement.

The objective of this project is to analyze the patient journey, identify bottlenecks, predict treatment completion, and recommend strategies to improve patient conversion and business performance.

---

## Project Objectives

* Perform data quality assessment.
* Analyze the patient conversion funnel.
* Identify patient drop-off stages.
* Evaluate provider performance.
* Analyze country-wise performance.
* Measure communication effectiveness.
* Analyze revenue and profitability.
* Build a machine learning model for treatment completion prediction.
* Segment patients based on engagement and risk.
* Generate actionable business recommendations.
